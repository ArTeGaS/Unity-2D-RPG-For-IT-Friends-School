# Unity 2D RPG Prototype

Unity 2D classroom prototype focused on basic RPG / action mechanics: player movement, projectiles, enemies, spawning, UI menus, and simple save-related code.

## Status

Educational prototype. The repository is useful for reviewing gameplay scripts and Unity project structure, but it is not a finished game or a production template.

## What is included

- Player camera and projectile controller scripts
- Enemy movement and enemy spawner scripts
- Main menu and settings menu UI scripts
- Save manager code
- Unity scene and project settings for local exploration
- Newtonsoft.Json package files used by the project

## Project structure

- `Assets/Player` - player-facing gameplay scripts
- `Assets/Enemies` - enemy behavior and spawning scripts
- `Assets/UI/MainMenu` - menu and settings UI scripts
- `Assets/Scenes` - Unity scenes

## Opening the project

This project was last worked with Unity `2022.3.12f1`.

Open the repository folder in Unity Hub, let Unity restore packages, then open `Assets/Scenes/SampleScene.unity`.

## Notes

This repository reflects a learning / teaching workflow. Some scripts and assets are intentionally simple and may need cleanup before reuse in a larger project.
